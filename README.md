##Regole di gioco: Bit&Blade
1)##Sistema di eroi: è possibile creare personaggi giocabili in base a 3 classi: Guerriero, Mago, Arciere
Ad ogni classe corrispondono statistiche iniziali diverse,a cui è possibile assegnare punti aggiuntivi al raggiungimento del livello successivo. 
Si sale di livello sconfiggendo nemici durante il combattimento, Ogni classe possiede spell ed equipaggiamenti unici.

2)##Combattimento: Viene scelto un nemico randomico tra nemici base(per semplicità solo goblin) e boss. Ogni 3 nemici casuali sconfitti il successivo sarà sempre un boss.
Il combattimento si svolge a turni, il calcolo del danno è influenzato sia dalle statistiche dell’eroe(compreso di equipaggiamento) sia da un moltiplicatore dipendente dal lancio di un dado. 
##Durante il proprio turno l’eroe può: 
-Usare un attacco base non consumando mana ma recuperarne una certa quantità
-Usare Spell con check del mana necessario, queste possono o danneggiare l’avversario,diminuire il suo attacco o potenziare l’attacco del proprio eroe
Alla vittoria di un combattimento verranno aggiornate le quest

3)##Quest: raggiunto un obiettivo l’eroe verrà ricompensato con un equipaggiamento. Gli obiettivi si distinguono in sconfitta nemici e raggiungimento livello

4)##Equipaggiamento: l’eroe può scegliere di equipaggiare un oggetto per ogni categoria (arma, armatura, accessorio). Ogni equipaggiamento può essere disponibile per tutti(Universale) o equipaggiabile solo dalla rispettiva classe influenzando così le statistiche finali
